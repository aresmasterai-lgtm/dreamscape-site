# ✅ Dreamscape Launch Checklist

Use this checklist to track your deployment progress.

---

## 📋 Pre-Launch

- [ ] GitHub account created & logged in
- [ ] Netlify account created (connected to GitHub)
- [ ] Domain purchased (if using custom domain)
- [ ] Google Analytics account set up (optional but recommended)

---

## 🚀 Deployment Steps

### GitHub Upload
- [ ] Opened terminal/command prompt
- [ ] Navigated to `dreamscape-site` folder
- [ ] Ran `git init`
- [ ] Ran `git add .`
- [ ] Ran `git commit -m "Initial commit"`
- [ ] Created GitHub repository
- [ ] Pushed code to GitHub

### Netlify Connection
- [ ] Clicked "Add new site" in Netlify
- [ ] Selected "Import from GitHub"
- [ ] Authorized GitHub access
- [ ] Selected `dreamscape-site` repository
- [ ] Clicked "Deploy site"
- [ ] Confirmed site deployed successfully
- [ ] Tested the Netlify URL

---

## 🎨 Configuration (Optional but Recommended)

### Google Analytics
- [ ] Created GA4 property
- [ ] Copied Measurement ID (G-XXXXXXXXXX)
- [ ] Updated `index.html` line 16 with real ID
- [ ] Pushed changes to GitHub
- [ ] Verified tracking in GA real-time report

### Custom Domain
- [ ] Added domain in Netlify
- [ ] Got DNS records from Netlify
- [ ] Added A record at domain registrar
- [ ] Added CNAME record for www
- [ ] Waited for DNS propagation (30 min - 48 hrs)
- [ ] Confirmed domain works with HTTPS

### Form Notifications
- [ ] Went to Netlify → Forms
- [ ] Set up email notifications for waitlist
- [ ] Set up email notifications for contact form
- [ ] Tested forms by submitting dummy data
- [ ] Confirmed email notification received
